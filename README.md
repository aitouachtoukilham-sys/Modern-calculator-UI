# Modern Calculator UI 🧮

A sleek and modern web calculator featuring a clean interface and a pastel color palette.

## 🚀 Technologies Used:
* *HTML5*: For the semantic structure.
* *CSS3*: For the modern styling, using custom colors and rounded corners.
* *JavaScript*: To handle calculations and logic.

## ✨ Key Features:
* *Elegant Design*: Focused on a minimalist and "classy" look.
* *Responsive Layout*: Works perfectly on desktops and mobile devices.
* *Interactive UI*: Soft button hover effects for a better user experience.

## 📂 Project Goal:
This project is part of my *Multimedia Development* portfolio, aiming to combine aesthetic design with functional code.
